# HockeyScoresAPI

💧 A project built with the Vapor web framework.

## Getting Started

### Building

To build the project, run the build script from the root of the project:
```bash
./build.sh
```

This script handles C++ SDK configuration needed for dependencies like swift-nio-ssl. If you prefer using `swift build` directly (Linux/other platforms), you can do so, but macOS may require the SDK configuration that's handled in the build script.

### Running

To run the project and start the server, use the following command:
```bash
swift run
```

### Testing

To execute tests, use the following command:
```bash
swift test
```
